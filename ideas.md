# Ideas de Diseño para Villa Bruni

## Objetivo
Crear un sitio web profesional y elegante para el alquiler de un apartamento de verano en la costa mediterránea española, que inspire confianza, transmita lujo accesible y facilite las solicitudes de reserva.

**Enfoque: Minimalismo Mediterráneo Contemporáneo**

**Movimiento de Diseño**: Inspirado en el minimalismo escandinavo fusionado con la calidez mediterránea, este enfoque busca espacios limpios con toques cálidos y orgánicos.

**Principios Fundamentales**:
1. **Claridad espacial**: Uso generoso de espacios en blanco para crear respiración visual
2. **Materialidad táctil**: Texturas sutiles que evocan arena, piedra caliza y lino
3. **Jerarquía cromática suave**: Paleta limitada con variaciones tonales sutiles
4. **Geometría orgánica**: Formas suaves y bordes redondeados que contrastan con líneas arquitectónicas limpias

**Filosofía de Color**:
- **Base**: Blanco cálido (off-white) y beige arena como fondos principales
- **Acentos**: Terracota suave (#D4A574) y azul mediterráneo desaturado (#7BA5B8)
- **Intención emocional**: Transmitir calma, calidez y sofisticación sin ostentación
- **Contraste**: Texto en gris carbón (#2C3338) para legibilidad óptima sobre fondos claros

**Paradigma de Layout**:
- **Estructura asimétrica con anclaje lateral**: Hero section con imagen a pantalla completa y texto superpuesto en tercio izquierdo
- **Grid fluido de 12 columnas** con rupturas intencionales para crear dinamismo
- **Secciones alternadas**: Contenido zigzag entre izquierda-derecha para mantener interés visual

**Elementos Distintivos**:
1. **Divisores orgánicos**: Transiciones entre secciones con curvas suaves (clip-path) que emulan dunas o olas
2. **Tarjetas flotantes**: Cards con sombras suaves y elevadas que parecen flotar sobre el fondo
3. **Iconografía lineal**: Íconos de trazo fino y minimalista para servicios y características

**Filosofía de Interacción**:
- **Transiciones fluidas**: Animaciones de 300-400ms con easing suave (ease-out)
- **Hover states sutiles**: Elevación ligera y cambio de opacidad en lugar de transformaciones dramáticas
- **Scroll reveals progresivos**: Elementos que aparecen con fade-in desde abajo al hacer scroll

**Animación**:
- **Entrada del hero**: Fade-in del texto con ligero desplazamiento vertical (20px)
- **Galería de imágenes**: Transiciones crossfade suaves entre imágenes
- **Formulario de reserva**: Validación con micro-animaciones (shake sutil en error, checkmark en éxito)

**Sistema Tipográfico**:
- **Display**: Cormorant Garamond (serif elegante) para títulos principales - peso 400/500
- **Body**: Lato (sans-serif limpia) para texto de lectura - peso 300/400
- **Jerarquía**: 
  - H1: 3.5rem (56px) / line-height 1.1
  - H2: 2.5rem (40px) / line-height 1.2
  - H3: 1.75rem (28px) / line-height 1.3
  - Body: 1.125rem (18px) / line-height 1.7

**Enfoque: Brutalismo Mediterráneo**

**Movimiento de Diseño**: Fusión del brutalismo digital con elementos mediterráneos tradicionales, creando un contraste audaz entre lo crudo y lo refinado.

**Principios Fundamentales**:
1. **Honestidad estructural**: Mostrar la "construcción" del sitio con bordes marcados y secciones claramente delimitadas
2. **Contraste dramático**: Yuxtaposición de elementos pesados con espacios negativos amplios
3. **Tipografía como arquitectura**: Texto grande y bold que funciona como elemento estructural
4. **Funcionalidad expuesta**: Elementos UI que no ocultan su propósito

**Filosofía de Color**:
- **Base**: Blanco puro (#FFFFFF) y negro profundo (#0A0A0A)
- **Acentos**: Azul cobalto intenso (#0047AB) y ocre mediterráneo (#CC8800)
- **Intención emocional**: Confianza, modernidad radical, memorable
- **Contraste**: Máximo contraste para legibilidad y impacto visual

**Paradigma de Layout**:
- **Grid rígido expuesto**: Sistema de cuadrícula visible con líneas divisorias marcadas
- **Bloques de contenido apilados**: Secciones de altura completa que se apilan verticalmente
- **Asimetría intencional**: Elementos descentrados con pesos visuales compensados

**Elementos Distintivos**:
1. **Bordes gruesos**: Borders de 3-4px en negro alrededor de elementos clave
2. **Tipografía monumental**: Títulos oversized que rompen el grid
3. **Botones sólidos**: CTAs con fondos sólidos y esquinas duras (border-radius: 0)

**Filosofía de Interacción**:
- **Transiciones abruptas**: Cambios instantáneos o muy rápidos (100ms)
- **Hover states contundentes**: Inversión de colores (negro↔blanco)
- **Feedback directo**: Sin sutilezas, respuestas visuales inmediatas

**Animación**:
- **Entrada del hero**: Slide-in desde arriba con timing duro
- **Scroll snapping**: Secciones que se anclan al viewport
- **Transiciones de página**: Wipe effects o cortes directos

**Sistema Tipográfico**:
- **Display**: Space Grotesk (sans-serif geométrica) para títulos - peso 700
- **Body**: IBM Plex Mono (monospace) para texto - peso 400/500
- **Jerarquía**: 
  - H1: 5rem (80px) / line-height 0.95
  - H2: 3rem (48px) / line-height 1.0
  - H3: 2rem (32px) / line-height 1.1
  - Body: 1rem (16px) / line-height 1.6

**Enfoque: Lujo Mediterráneo Sensorial**

**Movimiento de Diseño**: Inspirado en el diseño editorial de revistas de lujo y la fotografía arquitectónica de alta gama, este enfoque prioriza la experiencia visual inmersiva.

**Principios Fundamentales**:
1. **Fotografía como narrativa**: Imágenes de gran formato que cuentan la historia del lugar
2. **Capas de profundidad**: Uso de parallax y superposiciones para crear dimensión
3. **Elegancia discreta**: Lujo que se siente pero no se grita
4. **Ritmo visual**: Alternancia entre secciones densas y espacios de respiro

**Filosofía de Color**:
- **Base**: Crema cálido (#FAF8F3) y gris piedra (#E8E4DC)
- **Acentos**: Verde oliva profundo (#4A5D3F) y dorado antiguo (#B8956A)
- **Intención emocional**: Sofisticación, exclusividad, conexión con la naturaleza mediterránea
- **Contraste**: Texto en gris grafito (#3A3A3A) con detalles en dorado para elementos premium

**Paradigma de Layout**:
- **Sistema de revista editorial**: Layouts que recuerdan a spreads de revistas de arquitectura
- **Hero de pantalla completa con scroll reveal**: Primera impresión impactante con imagen inmersiva
- **Secciones con bleeding images**: Imágenes que sangran fuera de sus contenedores
- **Sidebar flotante para navegación**: Menú vertical elegante que aparece al scroll

**Elementos Distintivos**:
1. **Overlays con gradientes sutiles**: Capas semitransparentes sobre imágenes para mejorar legibilidad
2. **Líneas decorativas doradas**: Detalles lineales finos que enmarcan secciones importantes
3. **Números grandes estilizados**: Estadísticas o características mostradas con tipografía display oversized

**Filosofía de Interacción**:
- **Scroll storytelling**: Narrativa que se despliega mientras el usuario hace scroll
- **Parallax sutil**: Movimiento diferencial entre capas (velocidad 0.5x-1.5x)
- **Hover reveals**: Información adicional que aparece elegantemente al hover
- **Transiciones cinematográficas**: Fade y scale combinados para sensación premium

**Animación**:
- **Entrada del hero**: Zoom out lento de la imagen de fondo (scale 1.1 → 1.0) con fade-in del texto
- **Galería**: Efecto Ken Burns (zoom + pan lento) en imágenes
- **Scroll-triggered animations**: Elementos que se revelan con fade-in + translate-y mientras scrolleas
- **Loading states**: Skeleton screens con shimmer effect elegante

**Sistema Tipográfico**:
- **Display**: Playfair Display (serif clásica) para títulos - peso 400/600

## Selección Final

**Enfoque elegido: Lujo Mediterráneo Sensorial**

Este enfoque es el más adecuado para Villa Bruni porque:
- Transmite la experiencia premium de un apartamento de verano sin ser pretencioso
- La fotografía de gran formato permite mostrar las vistas y la ubicación privilegiada
- El estilo editorial genera confianza y profesionalismo
- Las animaciones cinematográficas crean una experiencia memorable
- La paleta de colores cálidos conecta emocionalmente con el Mediterráneo
- El sistema tipográfico combina elegancia (Playfair) con legibilidad moderna (Source Sans Pro)
